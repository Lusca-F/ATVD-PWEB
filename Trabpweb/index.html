Criação de página estática para atividade de pweb.

HTML é uma linguagem de marcação fundamental para a construção de páginas web. Ele permite estruturar o conteúdo de um site, organizando elementos como textos, imagens, links e tabelas, além de criar uma interface amigável ao usuário.
Essa linguagem é composta por tags (marcadores) que indicam ao navegador como exibir cada parte do conteúdo.

< head >O elemento HTML "head" providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.
< body >O elemento "body" do HTML representa o conteúdo de um documento HTML. è permitido apenas um por documento.
< br >O elemento HTML quebra-de-linha "br" produz uma quebra de linha em um texto (carriage-return).
< p >O elemento HTML "p" representa um parágrafo.
< meta >O elemento HTML "meta" define qualquer informação de metadados que não podem ser definidos por outros elementos HTML.
